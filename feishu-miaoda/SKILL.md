# 飞书妙搭操作指南

飞书妙搭是飞书（Lark）推出的 AI 驱动低代码/无代码应用构建平台。用户可通过自然语言对话或可视化拖拽快速构建 Web 应用，无需传统编程基础。

**平台入口**：https://miaoda.feishu.cn

## 核心开发模式

### 模式一：对话开发（推荐新手）

1. 在主页输入框描述需求（如"帮我做一个活动报名页面"）
2. AI 生成概要设计，确认后进入开发
3. 持续对话迭代：增加页面、修改样式、接入数据
4. 使用 `@` 引用内置工具（数据库、文件存储等）
5. 实时预览 → 满意后发布

**提示词技巧**：见 [references/ai-prompting.md](references/ai-prompting.md)

### 模式二：可视化开发（精细调整）

1. 切换到「可视化」模式
2. 从左侧组件面板拖拽组件到画布
3. 右侧属性面板配置样式、数据绑定、事件
4. 两种模式可随时互相切换

**组件分类**：布局（容器/栅格）、展示（文本/图片/表格）、表单（输入框/选择器/上传）、交互（按钮/弹窗/标签页）、图表（折线图/柱状图/饼图）

## 应用发布

| 方式 | 说明 |
|------|------|
| 公开链接 | 任何人可访问，无需登录 |
| 飞书工作台 | 企业内部访问，需管理员配置 |
| 嵌入分享 | 生成 iframe 嵌入其他页面 |

操作：右上角「发布」→ 选发布目标 → 配置权限 → 获取链接。修改代码后需重新发布才更新线上版本。

## 内置服务

### 数据库（PostgreSQL）

每个应用有独立 PostgreSQL 数据库实例。三种操作方式：
- **可视化建表**：数据库面板中新建表、添加字段（无需写 SQL）
- **SQL 编辑器**：直接执行 PostgreSQL SQL 语句
- **代码中调用**：通过妙搭数据库 API 在前端逻辑中操作数据

完整 SQL 语法和字段类型：见 [references/database.md](references/database.md)

### 文件存储 & 身份认证

详情见 [references/services.md](references/services.md)

## 导入已有项目

| 导入方式 | 适用场景 |
|----------|----------|
| 从多维表格导入 | 已有飞书多维表格数据，快速建应用 |
| 从 ZIP 文件导入 | 迁移已有 React 前端项目（≤50MB） |
| 从 Replit/Lovable/Google AI Studio 导入 | 其他平台项目迁移 |

详情见 [references/import-methods.md](references/import-methods.md)

## 自动化任务

支持定时触发、数据库变更触发、Webhook 触发，可执行数据库操作、发送飞书消息、调用外部 API。

详情见 [references/automation.md](references/automation.md)

## 导航与主题

在「应用设置 → 导航与主题」中配置：
- 导航类型：顶部导航栏 / 侧边导航栏 / 底部 Tab（移动端）
- 主题：主色调、字体、圆角、间距、暗色/亮色模式（全局生效）

## 日志排查

「开发界面底部日志面板」或「设置 → 日志」：

| 日志类型 | 内容 |
|----------|------|
| 运行日志 | 前端报错、console 输出 |
| 任务日志 | 自动化任务执行记录和结果 |
| 构建日志 | 发布过程输出 |
| 数据库日志 | SQL 执行记录 |
