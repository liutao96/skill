# 内置服务：文件存储 & 身份认证

## 文件存储

### 概述

妙搭内置对象存储服务，用于存储应用中产生的文件（图片、文档、音视频等）。

### 核心能力

- 文件上传（支持多种文件类型）
- 文件下载与访问（生成访问 URL）
- 文件列表管理（浏览、删除）
- 权限控制（公开访问 / 私有访问）
- 与应用「文件上传」组件直接集成

### 使用方式

**方式一（推荐）**：在可视化编辑器中添加「文件上传」组件，自动关联存储服务，无需手动配置。

**方式二**：在代码中调用妙搭文件存储 SDK/API，编程式操作：
- 上传文件 → 返回可访问的文件 URL
- 按路径列出已存储文件
- 按文件 ID 或路径删除
- 获取文件元数据（大小、类型、上传时间）

**方式三**：在「文件存储」管理面板中直接上传和管理文件。

### 注意事项

- 文件存储容量根据套餐有上限
- 私有文件访问需要鉴权 Token，不可直接通过 URL 访问
- 大文件建议使用分片上传
- 注意区分临时 URL（有效期限制）和永久 URL

## 通讯录与身份认证（飞书 SSO）

### 概述

妙搭集成飞书通讯录，提供飞书账号登录（OAuth 2.0）、组织架构查询、用户信息获取等能力。

### 身份认证流程

1. 在「应用设置」中开启「身份认证」，选择「飞书 SSO」
2. 配置回调 URL 和权限范围
3. 用户访问应用 → 跳转飞书 OAuth 授权页 → 用户确认 → 携带 Token 返回应用
4. 应用通过 Token 获取用户信息并建立会话

### 可获取的用户信息字段

| 字段 | 说明 |
|------|------|
| user_id | 飞书用户唯一 ID |
| name | 用户姓名 |
| email | 邮箱地址 |
| avatar_url | 头像图片 URL |
| department_id | 所属部门 ID |
| department_name | 所属部门名称 |
| job_title | 职位 |

### 通讯录查询能力

- 获取部门列表（支持树形结构）
- 获取部门下成员列表
- 根据用户 ID 查询详细信息
- 搜索用户（按姓名、邮箱）
- **人员选择器组件**：直接在表单中搜索和选择飞书通讯录成员

### 权限控制示例

```
// 基于飞书用户身份的权限控制
if (currentUser.department_id === 'hr_dept_id') {
  // 显示 HR 专属功能
}
```

### 注意事项

- 需要企业飞书管理员授权应用访问通讯录
- 外部用户（非本企业飞书账号）无法通过飞书 SSO 登录
- 通讯录数据查询遵循飞书可见性设置，部分成员可能不可查
- 用户 Token 有过期时间，需处理 Token 刷新逻辑
